# Orders API

A simple Spring Boot project for managing products, orders and order items.  
Built to practice real-world backend development, domain modeling and clean architecture.

## Tech Stack
- Java 21
- Spring Boot 3.4.12
- Maven
- JDK BigDecimal for financial calculations

## Domain Features (in progress)
- Product with validation rules
- OrderItem with total calculation
- Order aggregate (coming next)

## How to Run
```bash
./mvnw spring-boot:run
````
More features coming soon
